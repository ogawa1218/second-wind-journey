## プロジェクト概要

### Second Wind Journey MVP（Cloudflare Workers）
- LINE botバックエンド: Cloudflare Workers + Supabase (PostgreSQL)
- フロントエンド: LINE Messaging API + LIFF（テキスト入力ゼロ哲学）
- AIペルソナ「MASH」: 元100kg生還者、兄貴肌 tough-love コーチ、一人称「俺」、150字以内
- 食事分析: Google Gemini 1.5 Pro（マルチモーダル画像→JSON）
- コーチメッセージ: Claude API（fetch直接呼び出し）
- ウェアラブル連携: Fitbit / Garmin サイレント自動同期
- Stripeペナルティ: 週次ランサボり → 自動課金 → プール配分
- スコアリング: Math.js PFC + 行動序列スコア（睡眠30pt + 食事30pt + 運動40pt）

## 重要な制約・ルール

### セキュリティ
- `.env.local` は `.gitignore` 済み、絶対にコミットしない
- `GARMIN_EMAIL` / `GARMIN_PASSWORD` はコード直書き禁止
- `~/.garth/` トークンは1年有効

### 日付計算ルール
- **必ず `date` コマンドで実際の現在日付を取得する**。会話の記憶・推測から「今日」を決定してはならない
- 年が明示されていない場合は2026年を使用（計画期間: 2026-06-12〜2026-11-22）

### 勝田の教訓
本人が「もう少し行けた」「行けそう」と言っても、計画された距離・強度を超えた文章は書かない。前倒し・過負荷の要求にはブレーキ役を担う。
